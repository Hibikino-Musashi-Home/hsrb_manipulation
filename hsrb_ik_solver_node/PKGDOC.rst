=====================
Overview
=====================

提供機能
---------
* 台車位置制御誤差にロバストな非衝突のIK解を返すHSR-B/C向けプラグイン


=============================
Example
=============================

ノードの起動
-----------------------------------
以下のコマンドを実行します．

.. code-block:: bash

   roslaunch hsrb_ik_solver_node example.launch

クライアントの実行
-----------------------------------
以下のコマンドを実行します．

.. code-block:: bash

   python3 example/solve_ik.py
